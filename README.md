# AI Image Verification using Blockchain Watermarking

This repository contains prototype source code for a Master degree blockchain course project.

## Run Prototype

```bash
pip install -r requirements.txt
cd backend
python app.py
```

Open the local Flask URL in your browser.

## Notes

The current blockchain service is a mock service for simple demonstration. Replace it with Web3.py calls after deploying `smart_contract/ImageVerification.sol` to Ganache.
